# 🌤️ Modern Interactive Weather Dashboard

A sleek, responsive, and immersive weather application built with a premium modern dark-mode aesthetic. The dashboard provides real-time tracking, geolocation support, and beautifully visualized forecast data using fluid animations and interactive trends.

---

## ✨ Features

* **📍 Dynamic Geolocation & Search:** Automatically detects your current location on startup or lets you query any city globally using the Open-Meteo Geocoding API.
* **📈 Interactive Temperature Trends:** Uses **Chart.js** to render a custom, smooth cubic-bezier line graph showcasing max temperature trends.
* **📅 Unified Forecast Row:** Integrates synchronized weather condition icons and localized daily temperature markers directly under the tracking graph.
* **💎 Premium Glassmorphism UI:** Features an aesthetic backdrop-blur layout accented with smooth hover-lift card animations (`translateY`) and dynamic glow shadows.
* **📱 Immersive Mobile Optimization:** Engineered with fluid breakpoints to auto-stack components neatly on ultra-narrow viewports and mobile devices.
* **👤 Custom Profile Settings:** Includes a persistent sidebar drawer allowing users to rename profiles, backed by `localStorage` persistence.

---

## 🛠️ Tech Stack

* **Frontend:** Semantic HTML5, CSS3 Custom Variables & Flexbox/Grid
* **Charts Engine:** [Chart.js](https://www.chartjs.org/) (via CDN)
* **Data Core:** [Open-Meteo API](https://open-meteo.com/) (Current & 7-Day Forecast)
* **Typography:** Inter (Google Fonts)
